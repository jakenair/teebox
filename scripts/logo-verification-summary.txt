The MediaWiki pageimages API returns the top-of-infobox image. For golf
courses this is almost always a course PHOTO (aerial shot, clubhouse, a
single hole) rather than the club's logo. To avoid contaminating the
asset set with photos misrepresented as logos, we required one of these
strong-signal cues for acceptance:

  - URL ends in .svg or contains ".svg/" mid-path (Wikipedia thumb of an
    SVG source — almost always a logo on Wikipedia)

    or "insignia"
